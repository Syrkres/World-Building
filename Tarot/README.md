# Tarot Cards
A set of files for Major and Minor Arcana card descriptions.

**Major Card Tags include:**
---
fileType: Tarot<br>
tarotType: Major<br>
card: <br>
synopsis: <br>
---

**Minor Card Tags include:**
---
fileType: Tarot<br>
tarotType: Minor<br>
suite: SUITE<br>
card: Ace of SUITE<br>
synopsis: <br>
---
